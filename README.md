# AI Tutor

Персональный ИИ-репетитор для мобильного обучения.

## Возможности

- Красивый тёмный интерфейс с анимациями
- Регистрация и авторизация
- Профиль с фото и именем
- Чат с ИИ-репетитором (API будет подключён)
- Темы обучения (математика, физика, химия, и др.)
- Плавные анимации переходов
- Material Design 3

## Сборка APK

APK собирается автоматически через GitHub Actions при каждом пуше в main.

### Ручная сборка

```bash
flutter pub get
flutter build apk --release
```

## Установка

Скачайте APK из раздела [Releases](https://github.com/Sliva2010/nasdfhb/releases) и установите на устройство.

## Технологии

- Flutter 3.x
- Provider (state management)
- Image Picker
- Shared Preferences
- Material Design 3
- Кастомные анимации
